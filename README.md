# Protocreds

[![Gem Version](https://badge.fury.io/rb/protocreds.svg)](https://rubygems.org/gems/protocreds)
[![Build Status](https://secure.travis-ci.org/protocreds/protocreds-ruby.svg?branch=master)](https://travis-ci.org/protocreds/protocreds-ruby)
[![Apache 2 licensed](https://img.shields.io/badge/license-Apache2-blue.svg)](https://github.com/protocreds/protocreds-ruby/blob/master/LICENSE)

Protobuf-based credential format, inspired by ideas from Macaroons and
SPKI/SDSI.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'protocreds'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install protocreds
